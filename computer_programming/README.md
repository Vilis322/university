# *Computer Programming Exercises*

## *Description*
+ Educational Python exercises grouped by topic (APIs, data/time, OOP, iterators/generators, OS utilities, memory & decorators, vector math, etc.).
+ Each directory contains an independent, runnable mini-project with its own README.

## *Structure*
+ [**api**](api/) — small API utilities and loaders (I/O & endpoints).
+ [**bank_account**](bank_account/) — simple OOP model for accounts (balance operations, validation).
+ [**birthday_counter**](birthday_counter/) — count days until birthdays from a file.
+ [**datetime**](datetime/) — date/time utilities and mini-tasks.
+ [**generators**](generators/) — generator-based pipelines and lazy processing.
+ [**hero_class**](hero_class/) — OOP example with a `Hero` entity and actions.
+ [**iterators**](iterators/) — custom iterator patterns and protocol practice.
+ [**memory_management_and_decorators**](memory_management_and_decorators/) — references, mutability, decorators.
+ [**os**](os/) — interacting with filesystem, paths, and environment.
+ [**task_manager_project**](task_manager_project/) — task manager with tests, Makefile, and coverage.
+ [**vector**](vector/) — lightweight vector math class and operations.

## *Requirements*
+ **Python 3.10** or higher.
