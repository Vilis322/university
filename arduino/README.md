# *Arduino Projects*

## *Description*
+ Arduino/Tinkercad lab projects demonstrating digital I/O, sensors, servos, and serial communication.
+ Each subdirectory contains a `.ino` sketch, a wiring schema, and its own README.

## *Structure*
+ **binary_led_4bit/** — display a number (0-15) as 4-bit binary on LEDs.
+ **button_gestures/** — detect button press patterns (short, long, double-click).
+ **morse_led/** — blink Morse code on an LED.
+ **sequence_game/** — Simon-style memory game with LEDs and buttons.
+ **servo_lock_with_button/** — servo-controlled lock toggled by a button.
+ **seven_segment_button_counter/** — count button presses on a 7-segment display.
+ **ultrasonic_onepin_toggle/** — ultrasonic sensor single-pin toggle.
+ **ultrasonic_parking_assistant/** — distance-based parking LEDs and buzzer.
+ **ultrasonic_toggle/** — ultrasonic sensor toggle switch.

## *Requirements*
+ **Arduino IDE** or **Tinkercad**.
