# *Introduction to Programming*

## *Description*
+ Introductory Python exercises covering core language fundamentals.
+ Topics include conditionals, loops, string operations, lists, dictionaries, tuples, sets, functions, list comprehensions, exception handling, and standard library usage.

## *Structure*
+ **basic/** — standalone scripts for each topic (conditionals, loops, dicts, functions, exceptions, etc.).

## *Requirements*
+ **Python 3.10** or higher.
