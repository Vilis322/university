# *Software Testing*

## *Description*
+ Lab assignments for the Software Testing course (Java, JUnit, Selenium, Cucumber).

## *Structure*
+ **lab1/** — introduction to JUnit: basic assertions and test setup.
+ **lab2/** — Triangle classifier: equivalence partitioning, boundary value analysis, branch coverage (JaCoCo).
+ **lab3/** — Zoo Management System: user-story-driven tests with JaCoCo coverage.
+ **lab4/** — Cucumber + Selenium WebDriver: BDD scenario for the Student Registration Form.
+ **lab5/** — Python `unittest` + `coverage` on a Bank Account module.

## *Requirements*
+ **Java 17** or higher (lab1–lab4).
+ **JUnit 5** (lab1–lab3), **JUnit 4 + Cucumber 7** (lab4).
+ **Maven 3.9+**.
+ **Google Chrome** for lab4.
+ **Python 3.10+** for lab5.
