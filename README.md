# University

A collection of coursework, lab assignments, and exercises from university studies. Each directory corresponds to a separate course.

## Courses

| Directory | Language | Description |
|-----------|----------|-------------|
|[arduino](arduino/) | C++ (Arduino) | Arduino/Tinkercad lab projects — LEDs, sensors, servos, serial I/O |
| [computer_programming](computer_programming/) | Python | Applied Python: APIs, OOP, iterators, generators, decorators, Telegram bots |
| [discrete_mathematics](discrete_mathematics/) | Python | Number theory, prime algorithms, logical operations, continued fractions |
| [introduction_to_programming](introduction_to_programming/) | Python | Core Python fundamentals: conditionals, loops, strings, lists, dicts, functions |
| [mathematical_analysis](mathematical_analysis/) | Python | Numerical methods: integration, interpolation, Taylor series, ODE solvers |
| [oop](oop/) | Python | Object-oriented programming: class design, encapsulation, dataclasses |
| [physics](physics/) | Python | Physics simulations: oscillations, Brownian motion, optics, Planck's law |
| [software_testing](software_testing/) | Java | Software testing labs with JUnit 5 |
